HTML

  • The head element contains information abot the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag defines metadata about an HTML document.
  • The meta tags always go inside the head element.
  • The br tag creates a line break in text. It starts a new line without starting a new paragraph.
  • The word element and tag are used interchangeably. Element refers to the entire element and tag refers only to what appears inside the angle bracket.

Git

  • Git stands for global information tracker.
  • Git is a version control system that allows developers to work on projects.
  • Git lets you manage and keep track of each version of your project.
  • To access Git, we use the command line.
  • Git is great for tracking changes locally on your own computer
  • git status: checks what branch we are currenly on.
  • git checkout -b [branch-name]: creates a new branch and switches to it.
  • git add -A: add a file as it looks now in our working directory to the staging area of Git.
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot.
