HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The meta tag defines metadata about an HTML document.
  • The meta tags always go inside the head element.
  • The br tag creates a line break in text. It starts a new line without starting a new paragraph.
  • The word element and tag are used interchangeably. Element refers to the entire element and tag refers only to what appears inside the angle bracket.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource. The href attribute is using a relative path.

Git

  • Git stands for global information tracker.
  • Git is a version control system that allows developers to work on projects.
  • Git lets you manage and keep track of each version of your project.
  • To access Git, we use the command line.
  • Git is great for tracking changes locally on your own computer
  • git status: checks what branch we are currenly on.
  • git checkout -b [branch-name]: creates a new branch and switches to it.
  • git add -A: add a file as it looks now in our working directory to the staging area of Git.
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot.

Javascript

  • JavaScript is a programming language that can add interactivity to a website.
  • A variable is a named container that allows us to store data in our code.
  • The const keyword allows us to create variables that that can't be reassigned a value.
  • They let keyword allows us to reassign the value of the variable over time.
  • The console.log function is used to output a message to the web console.
  • Control flow is the order in which a computer executes code in a file. JavaScript is read from top to bottom, and order counts.
  • An array is a single variable that is used to hold a group of data. Arrays are typically used to hold data that is related in some way.
  • A Boolean value is a data type that can only have two values: true or false. Quotation marks are necessary for this value.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
